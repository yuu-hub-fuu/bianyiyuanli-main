# 🎨 Nexa Studio 现代化编译器IDE

> 从基础的tkinter应用升级为**专业级现代编译器IDE**

---

## 🚀 快速开始

```bash
cd bianyiyuanli-main
python -m nexa.ide.app
```

## ✨ 核心特性

### 1️⃣ **现代化UI设计**
- 采用VS Code Dark+ 配色方案
- 深灰/蓝色配色，低蓝光护眼
- 响应式交互（Hover/Active反馈）
- 专业的工具栏和菜单系统

### 2️⃣ **高效的编辑体验**
- **Ctrl+F / Ctrl+H** - 快速查找/替换，实时高亮匹配项
- **自动缩进** - Enter键自动继承缩进，行末`{`时额外缩进
- **括号配对** - 输入`{/(/[`自动补全，光标在中间
- **Tab转空格** - 按Tab自动插入4个空格
- **代码高亮** - 关键字、类型、字符串、注释着色

### 3️⃣ **强大的编译功能**
| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Enter` | 编译 |
| `F5` | 运行并跟踪 |
| `Ctrl+=/-/0` | 字体缩放（7-28pt） |
| `F12` | 跳转到定义 |

### 4️⃣ **完整的输出面板**
8个标签页全部可见：
- 🔤 **Tokens** - 词法分析
- 🌳 **AST** - 抽象语法树
- 📋 **Symbols** - 符号表
- 📝 **HIR** - 中间表示
- 🔀 **CFG** - 控制流图
- ⚡ **ASM** - x86-64汇编
- 🔷 **LLVM** - LLVM IR
- ⏱ **Timeline** - 编译流水线

### 5️⃣ **实时诊断**
- ❌ **错误** - 红色标记 + 快速修复建议
- ⚠️ **警告** - 黄色标记
- ℹ️ **信息** - 灰色标记
- 📍 双击诊断跳转到源代码

### 6️⃣ **美观的HTML报告**
- 现代暗色主题
- 左侧固定导航栏
- 顶部统计卡片
- 可折叠的代码块
- 代码复制按钮
- 时间轴流水线可视化

### 7️⃣ **国际化支持**
- 🇨🇳 中文菜单和对话框
- 🇺🇸 英文菜单和对话框
- 即时切换（Settings → Language）

## 🔧 高级用法

### 自定义配色
编辑 `nexa/ide/app.py` 第23行的颜色定义：

```python
BG       = "#1e1e1e"      # 修改背景色
ACCENT   = "#0078d4"      # 修改强调色
# ... 其他颜色
```

### 修改默认字体大小
编辑 `nexa/ide/app.py` 第180行：

```python
FONT_CODE_FALLBACK = ("Consolas", 13)  # 改为13
```

### 扩展菜单项
在 `_rebuild_menubar()` 方法中添加新菜单项。
